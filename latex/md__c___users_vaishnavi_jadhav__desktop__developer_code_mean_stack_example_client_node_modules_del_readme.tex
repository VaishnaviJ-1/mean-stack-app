The glob pattern {\ttfamily $\ast$$\ast$} matches all children and {\itshape the parent}.

So this won\textquotesingle{}t work\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{del.sync(['public/assets/**', '!public/assets/goat.png']);}

\end{DoxyCode}


You have to explicitly ignore the parent directories too\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{del.sync(['public/assets/**', '!public/assets', '!public/assets/goat.png']);}

\end{DoxyCode}


To delete all subdirectories inside {\ttfamily public/}, you can do\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{del.sync(['public/*/']);}

\end{DoxyCode}
