\begin{DoxyItemize}
\item URL parameters are case-\/insensitive
\item The {\ttfamily .host}, {\ttfamily .hostname} and {\ttfamily .port} properties cannot be set, and reading them does not return meaningful results (and are typed as {\ttfamily never}in Type\+Script)
\item The {\ttfamily .hosts} property contains a list of all hosts in the connection string
\item The {\ttfamily .href} property cannot be set, only read
\item There is an additional {\ttfamily .is\+SRV} property, set to {\ttfamily true} for {\ttfamily mongodb+srv\+://}
\item There is an additional {\ttfamily .clone()} utility method on the prototype
