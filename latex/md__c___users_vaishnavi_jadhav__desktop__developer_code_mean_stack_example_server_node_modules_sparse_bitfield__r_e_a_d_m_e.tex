Bitfield implementation that allocates a series of 1kb buffers to support sparse bitfields without allocating a massive buffer. If you want to simple implementation of a flat bitfield see the \href{https://github.com/fb55/bitfield}{\texttt{ bitfield}} module.

This module is mostly useful if you need a big bitfield where you won\textquotesingle{}t nessecarily set every bit.
