\href{https://github.com/csstools/postcss-plugins/tree/main/plugins/postcss-focus-visible}{\texttt{ Post\+CSS Focus Visible}} duplicates rules using the {\ttfamily \+:focus-\/visible} pseudo-\/class with a {\ttfamily .focus-\/visible} class selector, the same selector used by the \href{https://github.com/WICG/focus-visible}{\texttt{ focus-\/visible polyfill}}. This replacement selector can be changed using the {\ttfamily replace\+With} option.
