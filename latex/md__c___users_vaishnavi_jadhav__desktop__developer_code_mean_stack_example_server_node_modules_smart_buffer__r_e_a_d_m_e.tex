smart-\/buffer is a Buffer wrapper that adds automatic read \& write offset tracking, string operations, data insertions, and more.



{\bfseries{Key Features}}\+:
\begin{DoxyItemize}
\item Proxies all of the Buffer write and read functions
\item Keeps track of read and write offsets automatically
\item Grows the internal Buffer as needed
\item Useful string operations. (Null terminating strings)
\item Allows for inserting values at specific points in the Buffer
\item Built in Type\+Script
\item Type Definitions Provided
\item Browser Support (using Webpack/\+Browserify)
\item Full test coverage
\end{DoxyItemize}

{\bfseries{Requirements}}\+:
\begin{DoxyItemize}
\item Node v4.\+0+ is supported at this time. (Versions prior to 2.\+0 will work on node 0.\+10)
\end{DoxyItemize}\hypertarget{md__c___users_vaishnavi_jadhav__desktop__developer_code_mean_stack_example_server_node_modules_smart_buffer__r_e_a_d_m_e_autotoc_md18739}{}\doxysection{Breaking Changes in v4.\+0}\label{md__c___users_vaishnavi_jadhav__desktop__developer_code_mean_stack_example_server_node_modules_smart_buffer__r_e_a_d_m_e_autotoc_md18739}

\item Old constructor patterns have been completely removed. It\textquotesingle{}s now required to use the Smart\+Buffer.\+from\+XXX() factory constructors.
\item rewind(), skip(), move\+To() have been removed. (see offsets)
\item Internal private properties are now prefixed with underscores (\+\_\+)
\item {\bfseries{All}} write\+XXX() methods that are given an offset will now {\bfseries{overwrite data}} instead of insert. (see write vs insert)
\item insert\+XXX() methods have been added for when you want to insert data at a specific offset (this replaces the old behavior of write\+XXX() when an offset was provided)

With smart-\/buffer you don\textquotesingle{}t have to do either of those things.

\DoxyCodeLine{    packet.writeStringNT(country);}

In prior versions of Smart\+Buffer, .write\+XXX(value, offset) calls would insert data when an offset was provided. In version 4, this will now overwrite the data at the offset position. To insert data there are now corresponding .insert\+XXX(value, offset) methods.

\DoxyCodeLine{console.log(buff.toBuffer())}
\DoxyCodeLine{}
\DoxyCodeLine{// <Buffer 01 02 07 03 04 05 06>}

\end{DoxyCode}

\DoxyCodeLine{const buff = SmartBuffer.fromBuffer(new Buffer([1,2,3,4,5,6]));}

\DoxyCodeLine{console.log(buff.toBuffer());}
\DoxyCodeLine{}
