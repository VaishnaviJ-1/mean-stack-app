\item {\ttfamily BASIC=basic} becomes `\{BASIC\+: \textquotesingle{}basic'\}\`{}

\item empty values become empty strings ({\ttfamily EMPTY=} becomes `\{EMPTY\+: '\textquotesingle{}\}{\ttfamily )}
\item {\ttfamily inner quotes are maintained (think JSON) (}JSON=\{\char`\"{}foo\char`\"{}\+: \char`\"{}bar\char`\"{}\}{\ttfamily becomes}\{JSON\+:\char`\"{}\{\textbackslash{}\char`\"{}foo\textbackslash{}\char`\"{}\+: \textbackslash{}\char`\"{}bar\textbackslash{}\char`\"{}\}\char`\"{}{\ttfamily )}
