An sensible alternative to {\ttfamily Object.\+define\+Property()}. Available in npm and Ender as {\bfseries{prr}}.\hypertarget{md__c___users_vaishnavi_jadhav__desktop__developer_code_mean_stack_example_client_node_modules_prr__r_e_a_d_m_e_autotoc_md11462}{}\doxysection{Usage}\label{md__c___users_vaishnavi_jadhav__desktop__developer_code_mean_stack_example_client_node_modules_prr__r_e_a_d_m_e_autotoc_md11462}
Set the property `\textquotesingle{}foo'{\ttfamily (}obj.\+foo{\ttfamily ) to have the value}\textquotesingle{}bar\textquotesingle{}{\ttfamily with default options (}\textquotesingle{}enumerable\textquotesingle{}{\ttfamily ,}\textquotesingle{}configurable\textquotesingle{}{\ttfamily and}\textquotesingle{}writable\textquotesingle{}{\ttfamily are all}false\`{})\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{prr(obj, 'foo', 'bar')}

\end{DoxyCode}


Adjust the default options\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{prr(obj, 'foo', 'bar', \{ enumerable: true, writable: true \})}

\end{DoxyCode}


Do the same operation for multiple properties\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{prr(obj, \{ one: 'one', two: 'two' \})}
\DoxyCodeLine{// or with options:}
\DoxyCodeLine{prr(obj, \{ one: 'one', two: 'two' \}, \{ enumerable: true, writable: true \})}

\end{DoxyCode}
\hypertarget{md__c___users_vaishnavi_jadhav__desktop__developer_code_mean_stack_example_client_node_modules_prr__r_e_a_d_m_e_autotoc_md11463}{}\doxysubsection{Simplify!}\label{md__c___users_vaishnavi_jadhav__desktop__developer_code_mean_stack_example_client_node_modules_prr__r_e_a_d_m_e_autotoc_md11463}
But obviously, having to write out the full options object makes it nearly as bad as the original {\ttfamily Object.\+define\+Property()} so we can simplify.

As an alternative method we can use an options string where each character represents a option\+: `\textquotesingle{}e'==\textquotesingle{}enumerable\textquotesingle{}{\ttfamily ,}\textquotesingle{}c\textquotesingle{}==\textquotesingle{}configurable\textquotesingle{}{\ttfamily and}\textquotesingle{}w\textquotesingle{}==\textquotesingle{}writable\textquotesingle{}\`{}\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{prr(obj, 'foo', 'bar', 'ew') // enumerable and writable but not configurable}
\DoxyCodeLine{// muliple properties:}
\DoxyCodeLine{prr(obj, \{ one: 'one', two: 'two' \}, 'ewc') // configurable too}

\end{DoxyCode}
