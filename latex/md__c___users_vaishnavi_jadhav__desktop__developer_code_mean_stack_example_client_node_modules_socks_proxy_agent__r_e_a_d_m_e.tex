This module provides an {\ttfamily http.\+Agent} implementation that connects to a specified SOCKS proxy server, and can be used with the built-\/in {\ttfamily http} and {\ttfamily https} modules.

It can also be used in conjunction with the {\ttfamily ws} module to establish a Web\+Socket connection over a SOCKS proxy.
