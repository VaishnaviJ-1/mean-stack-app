Sends signal {\ttfamily signal} to all children processes of the process with pid {\ttfamily pid}, including {\ttfamily pid}. Signal defaults to {\ttfamily SIGTERM}.

For Linux, this uses {\ttfamily ps -\/o pid -\/-\/no-\/headers -\/-\/ppid PID} to find the parent pids of {\ttfamily PID}.

For Darwin/\+OSX, this uses {\ttfamily pgrep -\/P PID} to find the parent pids of {\ttfamily PID}.

For Windows, this uses `\textquotesingle{}taskkill /pid PID /T /F'\`{} to kill the process tree. Note that on Windows, sending the different kinds of POSIX signals is not possible.\hypertarget{md__c___users_vaishnavi_jadhav__desktop__developer_code_mean_stack_example_client_node_modules_tree_kill__r_e_a_d_m_e_autotoc_md14917}{}\doxysection{Install}\label{md__c___users_vaishnavi_jadhav__desktop__developer_code_mean_stack_example_client_node_modules_tree_kill__r_e_a_d_m_e_autotoc_md14917}
