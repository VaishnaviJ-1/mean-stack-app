Detect modules with circular dependencies when bundling with webpack.

Circular dependencies are often a necessity in complex software, the presence of a circular dependency doesn\textquotesingle{}t always imply a bug, but in the case where you believe a bug exists, this module may help find it.\hypertarget{md__c___users_vaishnavi_jadhav__desktop__developer_code_mean_stack_example_client_node_modules_cffe83c8ee709f5061c1ad83b7f4e73f3_autotoc_md2993}{}\doxysubsection{Webpack Versions}\label{md__c___users_vaishnavi_jadhav__desktop__developer_code_mean_stack_example_client_node_modules_cffe83c8ee709f5061c1ad83b7f4e73f3_autotoc_md2993}
The latest major version of this plugin {\ttfamily 5}, supports webpack {\ttfamily 4.\+0.\+1} and greater as a peer dependency.
