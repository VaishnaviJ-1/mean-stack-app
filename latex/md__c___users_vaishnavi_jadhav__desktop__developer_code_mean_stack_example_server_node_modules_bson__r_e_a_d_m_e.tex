BSON is short for \char`\"{}\+Binary JSON,\char`\"{} and is the binary-\/encoded serialization of JSON-\/like documents. You can learn more about it in \href{http://bsonspec.org}{\texttt{ the specification}}.

This browser version of the BSON parser is compiled using \href{https://rollupjs.org/}{\texttt{ rollup}} and the current version is pre-\/compiled in the {\ttfamily dist} directory.

This is the default BSON parser, however, there is a C++ Node.\+js addon version as well that does not support the browser.
