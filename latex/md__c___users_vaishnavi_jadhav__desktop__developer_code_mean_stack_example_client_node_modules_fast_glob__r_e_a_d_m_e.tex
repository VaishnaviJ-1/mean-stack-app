\begin{DoxyItemize}
\item An asterisk ({\ttfamily $\ast$}) — matches everything except slashes (path separators), hidden files (names starting with {\ttfamily .}).
\item A double star or globstar ({\ttfamily $\ast$$\ast$}) — matches zero or more directories.
\item Question mark ({\ttfamily ?}) – matches any single character except slashes (path separators).
\item Sequence ({\ttfamily \mbox{[}seq\mbox{]}}) — matches any character in sequence.
\end{DoxyItemize}

\begin{quote}

All patterns can be divided into two types\+:


\begin{DoxyItemize}
\item {\bfseries{static}}. A pattern is considered static if it can be used to get an entry on the file system without using matching mechanisms. For example, the {\ttfamily file.\+js} pattern is a static pattern because we can just verify that it exists on the file system.
\item {\bfseries{dynamic}}. A pattern is considered dynamic if it cannot be used directly to find occurrences without using a matching mechanisms. For example, the {\ttfamily $\ast$} pattern is a dynamic pattern because we cannot use this pattern directly.
\end{DoxyItemize}

A pattern is considered dynamic if it contains the following characters ({\ttfamily …} — any characters or their absence) or options\+:


\begin{DoxyItemize}
\item The \`{}case\+Sensitive\+Match\`{} option is disabled
\item {\ttfamily \textbackslash{}\textbackslash{}} (the escape character)
\item {\ttfamily $\ast$}, {\ttfamily ?}, {\ttfamily !} (at the beginning of line)
\item {\ttfamily \mbox{[}…\mbox{]}}
\item {\ttfamily (…$\vert$…)}
\item {\ttfamily @(…)}, {\ttfamily !(…)}, {\ttfamily $\ast$(…)}, {\ttfamily ?(…)}, {\ttfamily +(…)} (respects the \`{}extglob\`{} option)
\item {\ttfamily \{…,…\}}, {\ttfamily \{…..…\}} (respects the \`{}brace\+Expansion\`{} option)
\end{DoxyItemize}\hypertarget{md__c___users_vaishnavi_jadhav__desktop__developer_code_mean_stack_example_client_node_modules_fast_glob__r_e_a_d_m_e_autotoc_md5354}{}\doxysection{How to write patterns on Windows?}\label{md__c___users_vaishnavi_jadhav__desktop__developer_code_mean_stack_example_client_node_modules_fast_glob__r_e_a_d_m_e_autotoc_md5354}
