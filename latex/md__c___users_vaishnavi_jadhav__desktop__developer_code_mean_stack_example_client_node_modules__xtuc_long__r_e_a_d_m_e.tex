A Long class for representing a 64 bit two\textquotesingle{}s-\/complement integer value derived from the \href{https://github.com/google/closure-library}{\texttt{ Closure Library}} for stand-\/alone use and extended with unsigned support.

\href{https://www.npmjs.com/package/long}{\texttt{ }} \href{https://travis-ci.org/dcodeIO/long.js}{\texttt{ }}\hypertarget{md__c___users_vaishnavi_jadhav__desktop__developer_code_mean_stack_example_client_node_modules__xtuc_long__r_e_a_d_m_e_autotoc_md1475}{}\doxysection{Background}\label{md__c___users_vaishnavi_jadhav__desktop__developer_code_mean_stack_example_client_node_modules__xtuc_long__r_e_a_d_m_e_autotoc_md1475}
As of \href{http://ecma262-5.com/ELS5_HTML.htm\#Section_8.5}{\texttt{ ECMA-\/262 5th Edition}}, \char`\"{}all the positive and negative integers whose magnitude is no greater than 2$<$sup$>$53$<$/sup$>$ are representable in the Number type\char`\"{}, which is \char`\"{}representing the doubleprecision 64-\/bit format IEEE 754 values as specified in the IEEE Standard for Binary Floating-\/\+Point Arithmetic\char`\"{}. The \href{https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number/MAX_SAFE_INTEGER}{\texttt{ maximum safe integer}} in Java\+Script is 2\textsuperscript{53}-\/1.

Example\+: 2\textsuperscript{64}-\/1 is 1844674407370955$\ast$$\ast$1615$\ast$$\ast$ but in Java\+Script it evaluates to 1844674407370955$\ast$$\ast$2000$\ast$$\ast$.

Furthermore, bitwise operators in Java\+Script \char`\"{}deal only with integers in the range −2$<$sup$>$31$<$/sup$>$ through 2$<$sup$>$31$<$/sup$>$−1, inclusive, or in the range 0 through 2$<$sup$>$32$<$/sup$>$−1, inclusive. These operators accept any value of the Number type but first convert each such value to one of 2$<$sup$>$32$<$/sup$>$ integer values.\char`\"{}

In some use cases, however, it is required to be able to reliably work with and perform bitwise operations on the full 64 bits. This is where long.\+js comes into play.\hypertarget{md__c___users_vaishnavi_jadhav__desktop__developer_code_mean_stack_example_client_node_modules__xtuc_long__r_e_a_d_m_e_autotoc_md1476}{}\doxysection{Usage}\label{md__c___users_vaishnavi_jadhav__desktop__developer_code_mean_stack_example_client_node_modules__xtuc_long__r_e_a_d_m_e_autotoc_md1476}
