\href{https://github.com/csstools/postcss-plugins/tree/main/postcss-focus-within}{\texttt{ Post\+CSS Focus Within}} duplicates rules using the {\ttfamily \+:focus-\/within} pseudo-\/class with a {\ttfamily \mbox{[}focus-\/within\mbox{]}} attribute selector, the same selector used by the \href{https://github.com/jsxtools/focus-within}{\texttt{ focus-\/within polyfill}}. This replacement selector can be changed using the {\ttfamily replace\+With} option.
